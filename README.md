# primary-meal

primary meal source code

### git команды

Смотрим на изменения в коде

```bash
git status .
```

Добавляем все файлы

```bash
git add .
```

Коммитим файлы

```bash
git commit -m "initial commit"
```

Пушим файлы в репозиторий

```bash
git push --set-upstream origin dev
```

Меняем ветку

```bash
git checkout dev
```

Обновляем изменения

```bash
git pull
```
